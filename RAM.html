RAM minne

RAM minne är ett så kallat arbetsminne den lagrar minne medans program körs på datorn. Den sitter på moderkortet och lagrar data för att kunna ge data till processorn i stunden och eller inom kort. Den data som finns på RAM minnet när program körs lagras inte permanent som det gör i hårddisken. Allt detta sker efteroms RAM minnet kan arbeta oerhört mycket snabbare än hårddisken, och det går alltså fortare för processorn att hämta information från RAM minnet än att hämta det från hårddisken.
